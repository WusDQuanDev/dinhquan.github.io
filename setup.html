Hướng Dẫn Cài Đặt

Cài đặt trên iSH

Chạy các lệnh sau để cài đặt:

apk update
apk upgrade
apk add openssl python3 nodejs npm php
ln -s /usr/bin/python3 /usr/bin/python
npm install -g npm@latest
php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"
php composer-setup.php
php -r "unlink('composer-setup.php');"
mv composer.phar /usr/local/bin/composer

Cài đặt trên Termux

Chạy các lệnh sau để cài đặt:

pkg update && pkg upgrade
pkg install openssl python nodejs npm php
ln -s /data/data/com.termux/files/usr/bin/python3 /data/data/com.termux/files/usr/bin/python
npm install -g npm@latest
curl -sS https://getcomposer.org/installer | php
mv composer.phar $PREFIX/bin/composer
